ImgViewerPlus updated to version 0.5

Release notes
  • Added new feature: Moving image with right click(holding right mouse button).(You can access context menu by Shift+Right-click)
  • Added Flip functionality. 'H' key for flipping image horizontally and 'V' for flipping vertically.
  • Removed middle-click reset shortcut.(you can reset zoom by clicking on the image)
  • Increased move and zoom range.
  • Some code changes.
